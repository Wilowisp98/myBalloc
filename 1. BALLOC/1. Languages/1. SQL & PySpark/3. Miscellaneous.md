
### DESCRIBE

Shows basic metadata. (column names, data types)

SQL:
```sql
DESCRIBE [TABLE / SCHEMA]
```

---
### DESCRIBE EXTENDED

Shows more detailed metadata including storage information and statistics.

SQL:
```sql
DESCRIBE EXTENDED [TABLE / SCHEMA]
```
